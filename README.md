# Echelon Systems（エシュロンシステム）

企業のIT課題を解決する **IT Solutions & DX Partner** を想定した  
コーポレートサイトの模擬制作プロジェクトです。

デザインカンプをもとに、**保守性・再利用性を意識したコーディング**と  
**WordPressオリジナルテーマ構築**を前提として制作しています。

---

## 🔍 概要

- 種別：コーポレートサイト（模擬案件）
- 想定業種：ITソリューション / DX支援企業
- 目的：
  - サービス内容・強みの明確化
  - 問い合わせ獲得
  - 採用導線の整理

---

## 🛠 使用技術・構成

- HTML5
- CSS / SCSS（FLOCSS設計）
- JavaScript（GSAP / ScrollTrigger）
- WordPress（オリジナルテーマ）
- Gulp（開発環境）
  - Sassコンパイル
  - Autoprefixer
  - BrowserSync
  - 圧縮・整形

---

## 📐 設計方針

### CSS設計
- FLOCSS（Layout / Project / Component）
- 命名規則を統一し、引き継ぎやすさを重視
- 影響範囲が限定される構成

### JavaScript
- GSAPを使用したアニメーション実装
- ScrollTriggerによるスクロール連動演出
- 再利用可能な構造を意識

---

## ✨ 実装ポイント

- SVGロゴを使用したローディングアニメーション
- スクロールに応じたセクション演出
- ファーストビューでの視認性・メッセージ性を重視
- PC / SP 両対応（レスポンシブ）

---
## 📌 補足

本プロジェクトは **実在の企業・サービスとは関係ありません**。  
ポートフォリオ掲載を目的とした模擬制作です。
